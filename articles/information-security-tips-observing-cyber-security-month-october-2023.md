## Information Security Tips: Observing CYBER SECURITY Month

Information security is paramount at all times. Never share passwords. Inform users about the Information Security Policies. Keep screen locked and desk cleaned. Strong passwords, stored in Keeper. Do not allow use of a private PC/laptop for business purposes. Classify Information according to its security levels. Report incidents. Destroy printed classified documents after use. Never exchange user data by mail (e.g. data migration).
