## NICMAIL: PICKING UP THE PACE

NICMAIL or electronic mail on NICNET, is a facility which enables all the Users in the network to send &amp; receive messages and text files. NICMAIL links all the Xenix-based super-ATs with the NEC-S1000 at Delhi through satellite. This software package provides easy-to-use user interface to send and receive mail messages.

Apart from this, "Broadcasting" features, which allow the same mail to be sent to any number of NICNET Users at different geographical locations are also available.

The biggest advantage of NICMAIL is that the mail for a particular User is stored in an electronic mailbox. Thus an User can at any time log into the network and access the mail stored for him in his electronic mailbox.

NICMAIL is already proving to be indispensable for various government organizations. They are using NICMAIL not only for routine official instructions and mail, but also for transmitting urgent messages during emergencies.
