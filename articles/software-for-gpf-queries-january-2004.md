## Software for GPF Queries

A Web-enabled software ((http://www.agae.tn.nic.in) developed by NIC Tamil Nadu State Unit for the Office of the Accountant General (A&E), Chennai was recently launched . The software provides the following queries related to GPF to the State Government employees numbering about 7.0 lakhs :
a. Monthly credits / debits transactions for any year from 1991-92.
b. Current year Opening / Closing Balance credit
c. Opening / Closing Balance for any year since 1991-92
d. Missing credits (Year / Month)
e. Current year cumulative credits
f. Final withdrawal details like status, FW amount etc, for retired employees (along with nominee details, residual etc.

The GPF data is maintained for all transac- tions since 1991 and the total number of records in the database is about 100 million records. The software has been developed using SQL Server 2000 and ASP.NET.
