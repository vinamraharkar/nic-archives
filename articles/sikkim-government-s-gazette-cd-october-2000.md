## Sikkim Government's Gazette CD

A compact disk containing replica of whole 1999 Sikkim Government Gazettes (Ordinary and Extra ordinary) and a software to search of contents was released by the Hon'ble Governer of Sikkim, Sh. Randhir Singh on the occasion of State Day Celebration of Sikkim. The CD has been developed by NIC Sikkim State Unit in order to make the government gazettes portable. A user-friendly interface with standard features is the essence of the software. It enables keywords based search, and free text search, as well as querying dynamically on the gazette parameters. Animated tour has been provided to guide the user to access the information available on the CD. The software does not use any document search tools to implement the free text search. 32 bit windows applications with OLE automation feature have been incorporated to achieve this facility.
