## Treasury Accounting Software launched at Arunachal

With the pace of time, the workload in the treasuries of Arunachal Pradesh has increased manifold demanding efficiency and transparency in account work. In this context, a user friendly Treasury Accounting Software TAS 1.0 has been developed on request of the State Government by the software team of NIC, Arunachal Pradesh State Unit. Pilot phase implementation started in the Naharlagun sub-treasury from 1st July 2002. The system shall soon be implemented in other treasuries and sub-treasuries in a phased manner. This software is customized as per users’ requirement. It has stringent validation mechanism and various module level permissions. The software has been developed using Visual Basic at the Frontend and MS Access at the backend and is also available at http://offerings.nic.in
